TextCue AI - Solution for Install and Use Problems
==================================================

1. TextCue AI does not appear in Premiere Pro
--------------------------------------------
- Restart Premiere Pro after installation.
- Open Window > UXP Plugins > TextCue AI.
- Confirm Premiere Pro is version 25.6 or newer.
- Confirm Adobe Creative Cloud Desktop is installed and signed in.
- In Premiere Pro plugin preferences, enable Developer Mode if using a development build.
- Install again using the .ccx file or the UPIA command listed in README-WINDOWS.txt or README-MACOS.txt.

2. Windows installer says Adobe Unified Plugin Installer Agent is missing
------------------------------------------------------------------------
- Install or update Adobe Creative Cloud Desktop.
- Confirm this file exists:
  C:\Program Files\Common Files\Adobe\Adobe Desktop Common\RemoteComponents\UPI\UnifiedPluginInstallerAgent\UnifiedPluginInstallerAgent.exe
- If the path exists, run TextCueAIInstaller.exe as Administrator.

3. macOS command file will not open
----------------------------------
- Right-click the command file and choose Open.
- Or run this in Terminal from the macOS package folder:
  chmod +x "Install TextCue AI.command"
  ./Install\ TextCue\ AI.command

4. Double-clicking the .ccx file does nothing
--------------------------------------------
- Open Adobe Creative Cloud Desktop first, then double-click the .ccx again.
- Use UPIA manually:
  Windows:
    UnifiedPluginInstallerAgent.exe /install "FULL_PATH_TO\TextCue-AI-Premiere-Windows.ccx"
  macOS:
    ./UnifiedPluginInstallerAgent --install "FULL_PATH_TO/TextCue-AI-Premiere-macOS.ccx"

5. Panel opens but says no active sequence
-----------------------------------------
- Open a Premiere project.
- Open a sequence in the timeline.
- Click Scan Active Sequence again.

6. Transcript imports but no cues are found
------------------------------------------
- Use timestamped lines like:
  00:12 - At number 5, we have the TORVO TD3 Dash Cam.
  00:27 - It records in 4K with night vision.
  00:39 - The price is around $49.
- Confirm the Text Mode is Balanced or Heavy if you want specs and tutorial steps.

7. Create Text, Add Marker, or Jump does not work
-------------------------------------------------
- The MVP UI and detection logic are complete.
- Premiere Pro UXP host APIs for text-layer creation, marker metadata, playhead jumping, and undo removal can differ by Premiere build.
- Open src/premiere.js and update the TODO sections using the TypeScript definitions or API docs for the installed Premiere Pro version.
- Marker-only and text-creation payloads are already prepared in that adapter so the rest of the plugin does not need to change.

8. How to verify the plugin installed
------------------------------------
Windows:
  cd "C:\Program Files\Common Files\Adobe\Adobe Desktop Common\RemoteComponents\UPI\UnifiedPluginInstallerAgent"
  UnifiedPluginInstallerAgent.exe /list all

macOS:
  cd "/Library/Application Support/Adobe/Adobe Desktop Common/RemoteComponents/UPI/UnifiedPluginInstallerAgent/UnifiedPluginInstallerAgent.app/Contents/macOS"
  ./UnifiedPluginInstallerAgent --list all

9. Safe fallback for editors
----------------------------
- Paste or import transcript.
- Click Analyze Transcript.
- Use Export Report to create a review list.
- Use Add Markers Only when host marker APIs are wired for the installed Premiere version.
